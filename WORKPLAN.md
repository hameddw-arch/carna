# CARNA — خطة العمل التقنية الشاملة

> موقع إعلانات السيارات المبوبة — السوق السوري

---

## تحليل المشروع

### ما المشروع؟

منصة ويب سورية لبيع وشراء السيارات والتواصل مع خدماتها، تعمل كـ PWA لمواجهة ضعف الإنترنت، مع ربط كامل ببوابات الدفع والرسائل النصية المحلية.

### نقاط القوة في الوثيقة

- وعي حقيقي بالسوق المحلي (hosting سوري، دفع سوري، OTP محلي)
- تمييز واضح بين B2C (إعلانات) وB2B (ورشات)
- قرار صائب: المنصة وسيط فقط، لا تتدخل في الأسعار → يخفف المسؤولية القانونية
- PWA: ضروري بشكل استثنائي للسوق السوري

### فجوات يجب معالجتها

- لوحة تحكم الإدارة (Admin Panel) غير مذكورة — **ضرورة لا خيار**
- آلية مراجعة الإعلانات قبل النشر (Content Moderation) غائبة
- استراتيجية SEO غير موجودة — والإعلانات المبوبة تعيش على Google
- نظام التقييمات (Ratings) للبائعين والورشات غير مطروح
- لا ذكر لسياسة منع التكرار والإعلانات الوهمية

## القرارات التقنية المقترحة

| القرار | الخيار المختار | السبب |
| --- | --- | --- |
| Frontend | **React + Vite (PWA)** | نظام بيئي أكبر، دعم PWA عبر Vite PWA Plugin |
| Hosting Frontend | **Cloudflare Pages** | مجاني، سريع، مدمج مع Cloudflare — حساب موجود ✅ |
| Backend / API | **Cloudflare Workers** | Serverless، بدون VPS، مدمج مع Cloudflare |
| Database | **Supabase (PostgreSQL)** | حساب موجود ✅، قوي، Auth + Storage + Realtime مدمجة |
| Auth + OTP | **Supabase Auth + SMS** | يدعم phone OTP مع مزودي SMS المحليين |
| Real-time Chat | **Supabase Realtime** | مدمج، بدون Socket.io منفصل |
| File Storage | **Supabase Storage** | تخزين الصور مدمج، CDN تلقائي |
| CDN + DNS + SSL | **Cloudflare (Free)** | حماية DDoS، SSL تلقائي — حساب موجود ✅ |
| Domain | **Namecheap → carna.online** | محجوز ✅، Nameservers على Cloudflare ✅ |
| Email | **Resend** | Transactional API، إيصالات وإشعارات النظام |
| SMS Gateway | **MTN + Syriatel API** | OTP للمستخدمين السوريين، مزودان للضمان |

## بنية الشبكة والاستضافة

```
المستخدم (سوريا)
       ↓
  Cloudflare Edge  ✅ (حساب موجود + carna.online مربوط)
  ├── Pages   → React PWA (Frontend)
  ├── Workers → API / Business Logic (Serverless)
  ├── CDN     → صور وملفات ثابتة
  └── SSL/TLS تلقائي + DDoS Protection
       ↓
  Supabase  ✅ (حساب موجود)
  ├── PostgreSQL  → قاعدة البيانات
  ├── Auth        → تسجيل دخول + OTP بالهاتف
  ├── Realtime    → دردشة فورية بين البائع والمشتري
  └── Storage     → صور الإعلانات
```

> **الدومين:** `carna.online` — محجوز ✅ — Nameservers على Cloudflare ✅
>
> **البريد:** Resend — إيصالات، إشعارات Admin، تأكيد الإعلانات
>
> **لا VPS — لا سيرفر للإدارة — كل شيء Serverless**

## هيكل قاعدة البيانات (Database Schema)

### جداول المحور

```
users               → id, phone, name, avatar, verified, wallet_balance, created_at
listings            → id, user_id, title, description, price, city, status, views, created_at
listing_images      → id, listing_id, url, order
listing_tags        → id, listing_id, tag (e.g. #كيا, #دمشق, #2015)
services            → id, name, category, city, phone, description, verified, rating
service_categories  → id, name (ميكانيك / كهرباء / دهان / فحص فني ...)
messages            → id, sender_id, receiver_id, listing_id, body, read_at, created_at
notifications       → id, user_id, type, payload, read_at
wallet_transactions → id, user_id, amount, type, method, ref, created_at
ad_packages         → id, name, price, duration_days, features (JSON)
featured_listings   → id, listing_id, package_id, starts_at, ends_at
otp_codes           → id, phone, code, expires_at, used
legal_agreements    → id, user_id, version, agreed_at
```
